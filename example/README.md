# Chatbot UI

A customizable Flutter chat UI package with easy integration for local or API-based AI chatbots.

## Features

- 🎨 Fully customizable UI with themes
- 🤖 Support for multiple AI services (OpenAI, Local, Custom)
- 📝 Streaming responses support
- 💬 Typing indicators
- 🔄 Error handling and retry
- 🎯 Easy to integrate
- 📱 Responsive design

## Installation

Add to your `pubspec.yaml`:

```yaml
dependencies:
  chatbot_ui: ^1.0.0
